# ldap-server
LDAP Server Docker Stack 👤🐳

### Starting the LDAP Server

Navigate to this directory and execute:

```bash
docker-compose up
```

### Frontend

After you started the Server, you can navigate to [http://localhost:8008/](http://localhost:8008/) in your Browser.
